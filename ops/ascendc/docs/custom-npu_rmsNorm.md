# 应用场景说明
智谱GLM端侧模型基于CANN，已完成与昇腾和麒麟的适配。智谱在云侧昇腾环境中开发Ascend C自定义算子，完成模型微调与量化，并借助CANN工具链实现该模型的“一步直转”，算子代码直接复用，高效部署至基于麒麟芯片的端侧设备，支持在PC场景中快速落地。这种创新模式大幅降低了端云模型、算子适配工作量，也验证了CANN体系的开放性与研发效率。本方案rmsNorm算子在这个场景中得到验证和应用。

# RmsNormCustom自定义算子样例说明
本样例通过Ascend C编程语言调用RmsNorm高阶API，实现RmsNorm融合算子。
按照工程创建->算子实现->编译部署>算子调用的流程完成算子开发。整个过程都依赖于算子工程：基于工程代码框架完成算子核函数的开发和Tiling实现，通过工程编译脚本完成算子的编译部署，继而实现单算子调用或第三方框架中的算子调用。
