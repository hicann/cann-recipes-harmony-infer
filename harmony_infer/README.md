# 鸿蒙应用推理部署
算子开发和模型转换完成后，如果希望在鸿蒙设备上进行推理，需要调用NDK接口进行鸿蒙应用开发部署。

详细开发流程请参考[CANN Kit模型推理开发指南](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-model-inference)。
